### Hexadecimals:

10H = 16^1 = 16 x 16 = 100H
100H = 16^2 = 256 x 16 = 1000H
1000H = 16^3 = 4096 x 16 = 10000H
10000H = 16^4 = 65536 x 16 = 100000H
100000H = 16^5 = 1048576 x 16 = 1000000H
1000000H = 16^6 = 16777216 x 16 = 10000000H
